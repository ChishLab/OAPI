> **PHP 版本必须 >= 7.4 才能正常使用！**

## 开始

### 准备工作

```bash
$ git clone https://github.com/OAPI-Project/OAPI.git
$ mv OAPI-master oapi
$ cd oapi/
$ composer install
```

### 修改配置

1. 将 `项目目录/data/config/` 下的 `config-example.yaml` 复制一份并重命名为 `config.yaml`
2. 修改配置内容 (**其中数据库配置必须完善，否则无法启动**)
3. 最后，保存！

### 启动

**请确保在项目根目录下**

```bash
# Linux
$ ./start

# Windows
> php start
```
